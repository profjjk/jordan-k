My Job

This is a full stack web application that empowers a user to better organize their job search. The user can create a unique profile, add job applications, track their status, update with notes, and compare data.

Bookish

Bookish is a reading list application that allows you to keep a list of which books you have read, and it allows you to add new books that you want to read later.

GoT Quiz

This is a Game of Thrones themed quiz consisting of 20 questions and set to a timer. Wrong answers will decrease the timer. Correct answers will be scored and a percentage will display at the end. Player names and scores are stored for reference and can be called upon by clicking on the scoreboard.
